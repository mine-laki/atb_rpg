# EmojiParadigm — キャラクター強化システム仕様

## 強化の3系統

### 系統1: レベルアップ (Gil消費)
- レベル上限: 50
- 基礎ステータスを底上げ
- コスト: `100 × 1.18^(currentLevel - 1)` Gil

| キャラタイプ | HP成長/Lv | STR成長/Lv | MAG成長/Lv |
|-------------|----------|-----------|-----------|
| 前衛アタッカー | +60 | +5 | +2 |
| 魔法アタッカー | +40 | +2 | +5 |
| タンク | +80 | +4 | +2 |
| ヒーラー/支援 | +45 | +2 | +4 |
| 万能型 | +50 | +3 | +3 |

### 系統2: ロールレベルアップ (Gil + ロールクリスタル消費)
- ロールレベル上限: 10
- コスト: `base[role] × 1.5^(currentRoleLv - 1)` Gil + クリスタル×currentRoleLv個

| ロール | 基本コスト | Lv10ボーナス |
|--------|----------|------------|
| ATK | 200Gil | 物理ダメージ+30% |
| BLA | 200Gil | チェーンボーナス+30% |
| DEF | 180Gil | 被ダメ軽減+20% |
| HLR | 180Gil | 回復量+40% |
| ENH | 220Gil | バフ効果時間+50% |
| JAM | 220Gil | デバフ成功率+50% |

### 系統3: スキルボード (Gil + 素材消費)
- キャラ固有スキルツリー (5〜8ノード)
- ATBゲージ拡張・固有アビリティ解放・オートアビリティ追加
