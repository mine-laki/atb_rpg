# EmojiParadigm — ゲーム概要仕様

## プロジェクト情報

- **名称**: EmojiParadigm
- **ジャンル**: FF13風リアルタイムオートRPG
- **技術スタック**: React + TypeScript + Vite
- **公開先**: GitHub Pages (`gh-pages` ブランチ)

## ゲームフロー

```
タイトル
  └→ ホーム画面  ← 常にここに戻る起点
        ├→ 編成画面    (パーティ選択 / オプティマ登録)
        ├→ 強化画面    (キャラクター強化)
        ├→ ショップ    (装備購入 / 装備強化)
        └→ バトル画面  (ウェーブ制: 3連戦 → ボス)
              ├→ 勝利 → 報酬画面 (Gil + ドロップ表示) → ホームへ
              └→ 敗北 → リトライ or ホームへ
```

## 実装フェーズ

| Phase | 内容 |
|-------|------|
| P1 | 型定義・データ定義 (キャラ/敵/アビリティ/装備/素材) |
| P2 | ATBゲージ + チェーンゲージ システム |
| P3 | オートAI + ダメージ計算 |
| P4 | バトル画面UI (React) |
| P5 | オプティマパネル + 切替操作 |
| P6 | 編成画面 + 装備スロット表示 |
| P6.5 | セーブ/ロードシステム |
| P7 | 報酬・ドロップシステム |
| P7.5 | キャラクター強化画面 |
| P7.6 | ショップ画面 |
| P8 | ステージ・ボス・ゲームフロー整合 |
| P9 | 経済バランス調整 + GitHub Pages デプロイ |
