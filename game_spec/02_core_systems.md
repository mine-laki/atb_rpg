# EmojiParadigm — コアシステム仕様

## ATBゲージシステム

- キャラクターごとに 3〜6 本のATBセグメント
- 1セグメント = 3.0秒で充填
- エンハンサー(ヘイスト)時: ×1.5倍速
- ジャマー(スロウ)時: ×0.5倍速
- アビリティごとに ATBコスト 1〜4 を消費
- オプティマ切替時: 全員のATBゲージが 1セグメント分 即時回復

## チェーン＆ブレイクシステム

- チェーンゲージ: 0% 〜 999%
- 攻撃が止まると毎秒-15%減少
- アタッカー攻撃: +20% / hit
- ブラスター攻撃: +45% / hit
- 弱点属性ヒット: 増加量×1.5
- ブレイク閾値: 300%
- ブレイク中: ダメージ × チェーンボーナス倍率 (最大×9.99)
- ブレイク持続時間: 15秒
- ブレイク終了後: チェーンゲージ 0% にリセット

## ロールシステム (6種)

| ロール | 略称 | 絵文字 | 主な役割 |
|--------|------|--------|----------|
| アタッカー | ATK | ⚔️ | 高火力物理 / チェーン維持 |
| ブラスター | BLA | 🔮 | 属性魔法 / チェーン稼ぎ |
| ディフェンダー | DEF | 🛡️ | 敵ヘイト集中 / ダメージ軽減 |
| ヒーラー | HLR | 💚 | HP回復 / 蘇生 |
| エンハンサー | ENH | ✨ | 味方バフ |
| ジャマー | JAM | 💜 | 敵デバフ |

## オプティマ（作戦）システム

- 最大6種のオプティマを事前登録
- バトル中、ユーザーはオプティマ番号を選択して即時切替
- 切替後: 全員のロールが即座に変更 + ATBゲージが 1セグメント回復

## AIの行動優先度

```
ATK: HP>70% → ブレイバー > ラッシュ > たたかう
     HP<70% → たたかう (節約)
BLA: 弱点属性 → 最高コストアビリティ優先
     非弱点   → ファイラ or 通常魔法
DEF: 敵ヘイトが自分以外 → 挑発系
     常時      → ガード交えてたたかう
HLR: 誰かHP<50% → ケアルガ
     誰かHP<80% → ケアルラ
     全員HP満タン → ケアル (備え)
     誰か戦闘不能 → レイズ最優先
ENH: バフ未適用の味方 → バフ付与 (プロテス→ヘイスト→フェイス→シェル)
     全バフ済み → ケアル補助
JAM: デバフ未適用の敵 → デバフ付与
     全デバフ済み → 最強デバフ上書き or スロウ
```
